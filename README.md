# astrbot_plugin_smart_imagechat_hub

*_✨ [**AstrBot**](*https://github.com/AstrBotDevs/AstrBot*) 智能图片对话插件 ✨_*

LLM 驱动的 AstrBot 一体化智能图片对话插件。

自动为图库中的每张图片构建多重特征标签，支持手动传图 & 从群聊里偷图，并通过分析用户意图和对话语境，智能向用户回复合适的图片或表情包。

插件功能多样，可要求 bot 寻找想要的图片、在对话中发送符合语境的表情包等。未来将支持智能斗图。

## ⭐ 功能特性

- **多样化的图库构建**：支持手动上传 & 从群聊偷图，支持自动去重，支持可视化的图库管理。
- **自动图像特征标注**：上传图片后，调用 LLM 自动生成多个特征标签，辅助插件检索调用。
- **图像智能匹配**：基于 LLM 分析用户意图和对话语义，从图库中找出最符合需求的图片。
- **全新可视化 Pages**：提供功能强大的内嵌 Dashboard Page 管理中心，支持图库上传、批量筛选、标签编辑、进度查看、数据备份与恢复等，完整替代原生插件配置 WebUI。
- **备份恢复**：支持通过 Dashboard UI Page 进行插件数据的导入/导出，支持自动定时备份。
- **自定义文案**：自定义各种发图场景下的回复文案，支持固定文案或 LLM 接管回复。

## ▶️ 快速开始

### 第一步：配置图片转述模型

进入 AstrBot 的 **系统配置 > 提供商设置**，配置一个可用的聊天模型，并设置**默认图片转述模型**。该模型将用于自动为上传的图片生成特征标签。

### 第二步：进入 Dashboard UI Page 管理中心

点击 [**智能图片对话**] 插件卡片，打开 **插件行为 > 页面**，进入 `image-center-page`。

本页面基于 AstrBot 的新特性开发 (`>=4.24.2`)，无需配置端口映射等，一键直达。


![如何进入 Page UI](https://raw.githubusercontent.com/JREion/astrbot_plugin_smart_imagechat_hub/refs/heads/master/imgs/how_to_enter_page_ui.png)

### 第三步：上传图片 + 配置标签

1. **本地上传**：在 UI Page 中，点击 [**上传新图片**] 按钮。

   点击 [**选择文件**] 按钮选择本地图片，并点击右侧 [**上传图片**] 按钮上传到 AstrBot。支持 jpg、jpeg、png、gif、webp、bmp 格式。

2. **自动偷图**：在 UI Page 中，点击 [**自动收集群友发的图**] 按钮。

   填写 [**收集来源 QQ 群群号列表**]，以决定让 bot 从哪些群里偷图。

1. _(可选)_ 配置 [**公用特征标签**]：当需要为多张图片分配相同的标签时，可以预先填写。(公用标签不会直接应用到所有图像，需要在后续配置)

   <img src="https://raw.githubusercontent.com/JREion/astrbot_plugin_smart_imagechat_hub/refs/heads/master/imgs/upload_imgs_and_global_tags_in_page_ui.png" style="border: 2px solid gray;"/>

2. 上传图片后，插件会自动在后台调用 LLM 自动生成标签。可以在 UI Page 的进度面板中查看实时状态。

   <img src="https://raw.githubusercontent.com/JREion/astrbot_plugin_smart_imagechat_hub/refs/heads/master/imgs/import_imgs_then_check_tag_generation_step.png" style="border: 2px solid gray;"/>

### 第四步：在聊天中使用

本插件目前支持 2 种由 LLM 驱动的表情包回复能力：

1. **为用户寻找表情包**：

   在唤醒 bot (@机器人、使用唤醒词唤醒 bot、回复 bot、私聊等场景) 的对话中，包含特定关键词 ("表情包"、"照片" 等)，即可要求 bot 在图库里检索合适的图像，例如：

   ```
   @机器人 发一张开心的表情包
   @机器人 有没有可爱的美少女照片
   来一张 <bot 唤醒词> 的图片
   ```

   插件会根据用户描述，从图库中选择满足用户要求的图片发送。

2. **对话中主动发送表情包**：

   开启该功能后，bot 可以使用 LLM 分析自己发送的对话文本，并基于文本语义和图像标签，从图库里选取符合对话语境的表情包，发送给聊天用户。

未来考虑进一步添加 "智能斗图" 等新功能。

## ⚙️ 在 UI Page 中进行进一步配置

本插件同时维护 " Dashboard UI Page" 和 "AstrBot 标准插件配置页面" 两套前端。

UI Page 支持标准插件配置页面的全部设置。除此之外，它还支持图库预览、数据备份和恢复、配置实时保存等更多的便捷功能。

因此，我们推荐用户继续在 UI Page 中完成全部配置，无需进入标准插件配置页面。

### 1. 自动标签生成流程配置

点击 UI Page 图像处理进度栏中的 [**自动标签生成流程配置 ...**] 按钮，即可进行进一步配置。

| 配置项                       | 说明                                                         | 默认值       |
| ---------------------------- | ------------------------------------------------------------ | ------------ |
| 默认图片转述模型             | 默认继承 AstrBot 系统活动配置文件中的 “默认图片转述模型” 设置；保存后会同步写入系统配置。 | `[]`         |
| 预设标签类别                 | 指导 LLM 以什么标准作为自动标签生成的依据：图像类别 (是照片还是表情包)、人物特征、身材、情绪、动作、图片中的文本。 | `(全部勾选)` |
| 自定义标签类别               | 可输入多个自定义类别，各个类别之间可以用空格、逗号或顿号隔开。 | `“”`         |

### 2. 插件能力 - 为用户寻找表情包

| 配置项                      | 说明                                                         | 默认值                                               |
| --------------------------- | ------------------------------------------------------------ | ---------------------------------------------------- |

| 触发图片发送流程的关键词    | 只有用户唤醒机器人，且发送的文本中包含列表中的任意关键词，才会触发图像回复。 | `["照片", "表情包", "美图", "图片"]`                 |
| 找到图片后使用自定义文案    | 如果**开启**，则在发图的同时，输出固定的自定义文案；如果**关闭**，则会通过 LLM 回复，告知用户已经找到图片。 | `true`                                               |

### 4. 插件自动偷图

本插件支持从指定的 QQ 群中自动偷图，支持自动去重、人工&机器审核。工作流如下：

1. 插件从群聊中收集的图片，首先进入 [**待筛选图片池**]，此时不会被 LLM 打标签，也不会参与本插件的图库检索。
2. 用户或机器人，需要在 UI Page 中进行批量筛选，决定 [**待筛选图片池**] 中 丢弃 / 保留 哪些图片。
3. 保留的图片，会进入 [**固化图像库**]，按 "手动上传的图片" 相同的规则，参与打标签和检索。

<img src="https://raw.githubusercontent.com/JREion/astrbot_plugin_smart_imagechat_hub/refs/heads/master/imgs/filter_auto_collected_images.png" width="100%" style="border: 2px solid gray;"/>

点击 UI Page [**插件能力配置**] 栏中的 [**自动收集群友发的图**] 按钮，即可进行偷图行为的配置。

| 配置项                               | 说明                                                         | 默认值  |
| ------------------------------------ | ------------------------------------------------------------ | ------- |
| 启用自动收集图片功能                 | 开启后，插件会在指定 QQ 群中自动收集图片，放入 [待筛选图片池]。 | `true`  |
| 自动将所有待筛选图片入库             | 关闭时，用户可手动指定 [待筛选图片池] 中哪些图片存入 [固化图像库]。开启后，图像将实时自动入库。<br />***注意：开启本功能，可能存在无效表情包过多、使图像库爆满等风险。因此，除非群聊可信任，否则不建议开启。*** | `false` |
| 将入库图像加入插件其他功能的筛选流程 | 开启后，[固化图像库] 中的图片，会和手动上传图像一起，作为 “为用户寻找表情包” 和 “对话中主动发送表情包” 等插件图片功能的检索候选项。 | `true`  |
| 收集来源 QQ 群群号列表               | 每行一个 QQ 群号。机器人只在这些群聊中自动收集图片。         | `[]`    |
| 单张图片尺寸上限                     | 图像大小的上限。以 `KB` 作为计量单位。                       | `1024`  |
| 待筛选图片池存储数量上限             | 设置为 `-1` 表示无上限。                                     | `30`    |
| 自动拒绝曾经被丢弃过的图片           | 开启后，会记录在用户操作 [待筛选图片池] 时，被丢弃的图片信息；再次见到相同图片时直接跳过。 | `true`  |
| 固化图像库存储数量上限               | 设置为 `-1` 表示无上限。<br />***注意：如果开启了 [自动将所有待筛选图片入库] 功能，建议设置一个数量上限，以避免图库占用过多内存。*** | `300`   |

### 7. 插件数据导入/导出

页面顶部提供插件数据 导入/导出 功能：

- **导出配置**

  - 点击"导出配置"按钮，可在弹出的窗口中，选择需要的 ZIP 备份文件，或点击 [**手动导出**] 按钮，即可下载备份。包含：插件配置、图库索引、公用标签、图像标签、本地图片库。

  - 完成后，点击 "确认导入"，稍等片刻即可。

## ❓ 常见问题

**Q：上传图片后为什么没有自动生成标签？**

A：请确认已在 AstrBot 系统配置中设置了**默认图片转述模型**。可以在 UI Page 的进度面板中查看是否有报错信息。

**Q：标签生成失败了怎么办？**

A：可以在 UI Page 中点击"刷新"按钮重新触发标签生成，或检查图片转述模型是否正常工作。

**Q：AstrBot 版本号 `<4.24.2` 的用户，是否可以使用本插件？**

A：可以使用本插件中的**标准配置页面**，页面中包含了能使插件正常运行的全部功能。但是，标准配置页面无法实现数据备份、图库缩略图预览等功能，因此，我们建议您升级 AstrBot 版本。

**Q：如何只隐藏部分图片而不删除？**

A：在插件配置的 `hidden_images` 中添加图片的相对路径，图片将不会参与检索但保留在磁盘中。

**Q：自定义回复文案支持哪些变量？**

A：支持以下占位符：

- `{filename}` - 匹配图片的文件名
- `{tags}` - 图片的特征标签（顿号分隔）
- `{need}` - 用户需求摘要
- `{reason}` - 匹配理由

**Q：如何备份和迁移图库？**

A：使用 UI Page 中的"导出配置"功能，会生成包含所有数据的 ZIP 文件。在新环境中使用"导入配置"即可恢复。
